# Quiz: Introduction to NVIDIA Isaac Sim

Test your knowledge on the NVIDIA Isaac Sim robotics simulation platform.

---

## Question 1

NVIDIA Isaac Sim is built on which platform, leveraging physically accurate simulation and rendering?

a) Unity
b) Unreal Engine
c) Omniverse
d) Gazebo

**Answer**: c) Omniverse

---

## Question 2

Which of the following is a key feature of Isaac Sim that is particularly beneficial for training AI models by generating labeled data?

a) Real-time Factor (RTF)
b) Standard Definition Graphics (SDG)
c) Synthetic Data Generation (SDG)
d) Universal Robot Description Format (URDF)

**Answer**: c) Synthetic Data Generation (SDG)

---

## Question 3

What is "Warp" in the context of Isaac Sim's advantages for Reinforcement Learning?

a) A new communication protocol for ROS 2.
b) A GPU-accelerated physics engine for parallel simulations.
c) A feature for time-traveling in simulations.
d) A type of sensor noise model.

**Answer**: b) A GPU-accelerated physics engine for parallel simulations.

---

## Question 4

Isaac Sim's ability to easily randomize simulation parameters during training, such as textures and lighting, is known as what technique?

a) Texture Mapping
b) Domain Randomization
c) Environment Hashing
d) Scene Adaptation

**Answer**: b) Domain Randomization

---

## Question 5

Isaac Sim provides seamless integration with ROS 2 through:

a) Direct hardware control drivers.
b) A built-in ROS 2 bridge for data exchange.
c) A proprietary communication protocol.
d) Its own internal messaging system that replaces ROS 2.

**Answer**: b) A built-in ROS 2 bridge for data exchange.
