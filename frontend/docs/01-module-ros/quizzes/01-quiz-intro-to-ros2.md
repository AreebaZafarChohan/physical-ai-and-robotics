# Quiz: Introduction to ROS 2

Test your knowledge on the fundamentals of ROS 2.

---

## Question 1

What is the primary communication mechanism that enables loose coupling between nodes in ROS 2?

a) Services
b) Actions
c) Topics
d) Parameters

**Answer**: c) Topics

---

## Question 2

Which of the following is NOT a primary advantage of ROS 2 over ROS 1?

a) Real-time capabilities
b) Support for multi-robot systems
c) Simplified C++ syntax
d) Enhanced security features

**Answer**: c) Simplified C++ syntax (While improvements were made, "simplified C++ syntax" is not listed as a primary advantage in the context of architectural changes like DDS, security, and multi-robot support.)

---

## Question 3

A ROS 2 node that requests a specific computation and waits for a response is using which communication pattern?

a) Publish/Subscribe
b) Request/Reply
c) Goal/Feedback/Result
d) Get/Set

**Answer**: b) Request/Reply

---

## Question 4

NVIDIA Isaac Sim is built on which platform, leveraging physically accurate simulation and rendering?

a) Unity
b) Unreal Engine
c) Omniverse
d) Gazebo

**Answer**: c) Omniverse

---

## Question 5

What does the acronym "VLA" stand for in the context of Physical AI and Humanoid Robotics?

a) Virtual Learning Agent
b) Vision-Language-Action
c) Voice-Linked Autonomy
d) Visual Localization Algorithm

**Answer**: b) Vision-Language-Action
